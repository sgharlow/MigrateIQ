## MigrateIQ Chat Rules

When responding to questions about this project:

- This is a WideWorldImporters database application running on Microsoft SQL Server (T-SQL)
- The database uses advanced MSSQL features: temporal tables, native compilation,
  Row-Level Security, full-text search, geography types, JSON operations
- Application code is TypeScript using the `mssql` npm package
- When asked about migration, refer to the MigrateIQ flow which handles automated
  MSSQL to PostgreSQL translation
